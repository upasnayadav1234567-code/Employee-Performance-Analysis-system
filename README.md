# Employee Performance Analytics Dashboard

## Project Overview

The Employee Performance Analytics Dashboard is a Data Analytics project developed using Python. The project demonstrates the complete data analysis workflow, including data loading, data cleaning, exploratory data analysis (EDA), statistical analysis, and data visualization.

The dataset contains employee information such as department, salary, tenure, and performance scores. The project aims to identify performance trends, analyze departmental performance, and discover relationships between employee experience, salary, and performance.

---

## Objectives

* Load and explore employee data using Pandas.
* Handle missing values and clean the dataset.
* Remove duplicate records.
* Perform statistical analysis on employee performance.
* Analyze department-wise performance metrics.
* Identify top-performing employees.
* Measure correlations between salary, tenure, and performance.
* Generate professional visualizations for business insights.

---

## Project Workflow

### Step 1: Data Loading & Exploration

* Import employee dataset.
* Display dataset structure.
* Check dataset dimensions.
* Identify missing values.
* Generate descriptive statistics.

### Step 2: Data Cleaning

* Replace missing values using median imputation.
* Remove duplicate records.
* Standardize data types.

### Step 3: Data Analysis

* Calculate average performance by department.
* Identify top-performing employees.
* Analyze salary-performance relationships.
* Analyze tenure-performance relationships.
* Calculate correlation coefficients.

### Step 4: Data Visualization

The project generates the following visualizations:

1. Performance Score Distribution
2. Department-wise Average Performance
3. Salary vs Performance Scatter Plot
4. Performance vs Employee Tenure
5. Correlation Matrix Heatmap

### Step 5: Executive Report

A summary report is generated highlighting key findings and business insights.
